- [x] 实现 `ai_client.py`（AI客户端）

#### 核心接口
```python
# TRIZ引擎
from src.core.triz_engine import get_triz_engine
engine = get_triz_engine()
session = engine.analyze_problem(problem="...", use_ai=False)

# 矛盾矩阵
from src.core.matrix_selector import get_matrix_manager
matrix = get_matrix_manager().get_matrix("39")
principles = matrix.find_solutions("速度", "能耗")

# 本地存储
from src.data.local_storage import LocalStorage
storage = LocalStorage()
storage.save_session(session)
```
