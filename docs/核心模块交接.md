## 🔑 关键接口

### TRIZEngine使用
```python
from src.core.triz_engine import get_triz_engine

engine = get_triz_engine()

# 分析问题
session = engine.analyze_problem(
    problem="手机需要更大电池但要保持轻薄",
    improving_param="移动物体用的能源",  # 可选
    worsening_param="运动物体的体积",       # 可选
    use_ai=False  # 默认关闭
)

# session包含：
# - session.problem          # 问题描述
# - session.solutions       # 解决方案列表
# - session.improving_param  # 改善参数
# - session.worsening_param # 恶化参数
# - session.ai_enabled      # AI是否启用
```

### 矛盾矩阵使用
```python
from src.core.matrix_selector import get_matrix_manager

manager = get_matrix_manager()
matrix = manager.get_matrix("39")

# 查询矩阵
result = matrix.query_matrix("强度", "重量")
print(result.principle_ids)  # [1, 8, 40]

# 获取所有参数
params = matrix.get_improving_params()
```

### 本地存储使用
```python
from src.data.local_storage import LocalStorage

storage = LocalStorage()
storage.initialize()

# 保存会话
storage.save_session(session)

# 获取历史
sessions = storage.get_sessions(limit=50)
```

## ⚠️ 注意事项

### 1. AI开关状态
- AI功能**默认关闭**
- 用户必须**主动开启**
- AI需要网络连接

### 2. 参数选择
- 改善参数和恶化参数都是**可选**的
- 如果留空，系统会自动检测
- 参数检测使用关键词匹配算法

### 3. 解决方案数量
- 可以设置为**0**（0=不给方案）
- 建议最大值限制在**20**以内

### 4. 数据持久化
- 使用SQLite数据库
- 数据库文件：`triz_sessions.db`
- 启用WAL模式提高性能
